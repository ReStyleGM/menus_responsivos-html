# menus_responsivos-html
Templates de menu responsivo contendo HTML, CSS e JS como base para a criação. Os menus tem diversas complexidades.

# GERAL
- Serão disponibilizados diversos menus responsivos, alguns mais simples e outros mais completos e complexos. São utilizados HTML 5, CSS, JS para a realiazação dos códigos demonstrados nessa parte;
- Os codigos são compatíveis com celulares, computadores e tablets em todos os casos demonstrados e caso queira aplicar mudanças de coloração, os códigos terão disponíveis as mudanças;
- Alguns utilizei material externo  com uso de inspiração, mas fiz 100% a mão e do zero todos eles.

- A intenção dessa sessão é treinamento de código e auxiliar, sendo todos códigos livre to use.
